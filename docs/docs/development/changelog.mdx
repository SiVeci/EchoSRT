# 📝 更新日志

本页记录 EchoSRT 各版本的显著变更。

---

## v1.1.1 (2026-05-11)

### 🏗️ 架构级重构
- **多进程隔离**：将 Whisper 推理迁移至独立的 **Worker 子进程**，彻底解决模型卸载引发的 Python 闪退问题。
- **极致资源回收**：子进程空闲 5 分钟后自动销毁，由操作系统内核强制释放显存。

### 🤖 翻译引擎增强
- **DeepSeek-R1 适配**：支持用户自定义 **Max Tokens** 和 **Temperature**，解决长文本思考导致的截断问题。

### 🎨 UI 与交互修复
- **实时反馈自愈**：修复原声/翻译图标点亮延迟；引入 1.5s 安全延迟解决 NTFS 文件系统同步竞态。
- **资产查询加速**：禁用前端资产轮询的浏览器缓存。

### 🛠️ 系统健壮性
- **删除逻辑优化**：删除模型前强杀 Worker 进程释放文件锁。
- **安全防护补齐**：在所有入口添加 `freeze_support()` 以适配 Windows 多进程环境。

---

## v1.1.0 (2026-05-10)

- **方案池管理**：新增 API 配置多方案池，支持一键切换、克隆与平滑迁移。
- **媒体库加速**：重构底层扫描算法，大幅提升超大容量媒体库扫描速度。
- **下载链路重构**：统一全局资产下载行为，支持实时进度反馈。
- **配置保护**：确立后端文件为唯一真相源，防止配置覆盖。

---

## v1.0.1 (2026-05-08)

- **文档中心**：集成 Docusaurus 知识库。
- **安全性**：实现路径净化与 LLM 返回值拦截校验。

---

## v1.0.0 (2026-05-07)

- **架构重写**：针对低内存设备优化，引入后端模块化与异步并发队列。
- **物理切片**：引入基于 FFmpeg 的音频物理切片技术。
- **全局代理**：增加全局网络代理配置支持。
