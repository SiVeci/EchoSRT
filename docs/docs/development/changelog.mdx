# 📝 更新日志

本页记录 EchoSRT 各版本的显著变更，数据同步自 GitHub Releases。

---

## v1.0.1 (2026-05-08)

### 🚀 新功能与改进
- **文档中心**：集成 Docusaurus 知识库（即本 Wiki）并添加 WebUI 入口。
- **安全性增强**：实现文件名路径净化及 LLM 返回值拦截校验，提升系统健壮性。
- **性能优化**：重构 FFmpeg 进程管理机制，支持自动清理临时文件。
- **代码净化**：移除冗余代码，补全缺失的依赖声明。

---

## v1.0.0 (2026-05-07)

### 🏗️ 架构重构
- **内存优化**：针对低内存设备进行架构重写，通过后端模块化和异步并发队列彻底解决 OOM 问题。
- **物理切片**：引入基于 FFmpeg 的物理切片技术，提升超大文件处理能力。

### ✨ 新特性
- **代理支持**：增加全局代理配置。
- **流量控制**：实现 LLM 翻译流控制及本地模型生命周期管理。
- **用户体验**：支持任务拖拽排序，优化 Logo/Favicon 视觉展示，修复 Docker 部署问题。

---

## v0.9.2 (2026-05-04)

- **模块化**：将 `app.py` 拆分为模块化层级，并建立独立的配置目录。
- **性能**：引入并发翻译加速功能。
- **网络**：实现细粒度的代理路由，新增全局设置面板及核心节点的独立代理开关。

---

## v0.9.1 (2026-04-29)

- **部署**：新增 Docker 支持（提供 CPU 和 GPU 两种模式）。
- **稳定性**：在任务开始前增加代理连通性检查。

---

## v0.9.0 (2026-04-28)

- **后端**：引入异步流水线和后台 Worker 机制。
- **前端**：使用 Element Plus 重构 WebUI，新增实时 WebSocket 终端。
- **功能**：支持批量工作流、多文件上传队列及任务资产管理。

---

## v0.8.2 (2026-04-28)

- **网络**：支持全局 HTTP/SOCKS5 代理。
- **命令行**：更新 `main.py` 以支持高级参数。
- **启动**：通过 `start.py` 统一入口，取消强制性的虚拟环境检测。

---

## v0.8.1 (2026-04-27)

- **UI**：视觉重构引擎选项卡，增加激活状态指示器。
- **修复**：修复了本地 `faster-whisper` 初始化失败的 Bug。

---

## v0.8.0 (2026-04-27)

- **引擎**：引入云端 API 识别引擎（兼容 OpenAI 标准）。
- **特性**：支持说话人分离、词级时间戳，以及针对大文件（>24MB）的自动音频切片。
- **优化**：增加工具提示（Tooltips）并为 LLM 翻译配置结构化 Prompt 约束。

---

## v0.7.0 (2026-04-26)

- **品牌**：项目正式更名为 **EchoSRT**。
- **布局**：将全局控制项移动至顶部，新增历史记录和参数配置的折叠面板。
- **规范**：标准化排版和组件对齐。

---

## v0.6.0 (2026-04-26)

- **LLM**：支持动态模型列表获取，解耦目标语言选择。
- **工作流**：新增独立的“仅提取音频”工作流。
- **UI**：所有功能标签页采用卡片式布局。

---

## v0.5.0 (2026-04-26)

- **历史管理**：新增任务状态管理（加载/删除历史任务）。
- **设计**：引入卡片式组件设计，提升空间利用率。

---

## v0.4.0 (2026-04-26)

- **核心**：深度模块化改造，引入基于状态机的任务引擎。
- **智能翻译**：集成基于 LLM 的智能翻译（支持 DeepSeek, ChatGPT 等）。
- **FFmpeg**：开放高级 FFmpeg 参数配置。
- **UX**：全新的分栏式工作台设计。

---

## v0.3.0 (2026-04-26)

- **语言模型**：动态支持 99 种语言及全系列 `faster-whisper` 模型。
- **反馈**：新增上传、FFmpeg 处理和模型下载的实时进度条。
- **交互**：增加内联参数定义与工具提示。

---

## v0.2.0 (2026-04-25)

- **技术栈**：引入 FastAPI + Vue 3 前端架构。
- **管理**：支持工作空间自动清理和 GPU 显存自动释放（闲置 5 分钟后）。
- **稳定性**：显著提升 WebSocket 连接稳定性。

---

## v0.1.0 (2026-04-25)

- **首次发布**：核心 `faster-whisper` 工作流落地。
- **功能**：视频转音频提取、模型推理、SRT 字幕生成。
- **硬件**：支持 NVIDIA CUDA (float16) 自动检测及 CPU (int8) 降级运行。
- **环境**：本地化的 FFmpeg 路径自动解析。
