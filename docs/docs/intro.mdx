# 🎬 EchoSRT — 字幕工作台

**EchoSRT** 是一套为创作者、研究者与字幕组打造的视频翻译字幕工具链。它以 **FFmpeg + faster-whisper + LLM** 为核心引擎，提供从视频上传到字幕下载的一站式自动化流水线。

---

## 🌟 核心功能

| 模块 | 能力 | 技术实现 |
|------|------|----------|
| 🎵&nbsp;**音频提取** | 从视频中提取 16kHz/mono WAV，支持音轨选择与时间段裁剪 | FFmpeg (`-map`, `-ss`, `-to`) |
| 🎤&nbsp;**语音识别** | 本地 faster-whisper (CTranslate2) + 云端 OpenAI 兼容 API 双引擎 | `whisper_engine.py` / `api_transcribe.py` |
| 🌍&nbsp;**字幕翻译** | LLM 异步并发翻译，支持上下文拼接防断档 | `translate.py` (asyncio + Semaphore) |
| 📡&nbsp;**实时监控** | WebSocket 全双工通信与状态同步，流式传输进度可视化 | `ws_manager.py` |
| 🐳&nbsp;**一键部署** | 基于 Docker Compose 的标准化部署，支持 CUDA/NVIDIA GPU 加速 | `docker-compose.yml` + `Dockerfile.gpu` |

---

## 🏗️ 技术栈一览

| 架构层 | 技术选型 |
| :--- | :--- |
| **前端** | React + Capacitor |
| **后端** | FastAPI (Python 3.10+) + asyncio |
| **引擎** | FFmpeg + faster-whisper (CTranslate2) + OpenAI SDK |
| **通信** | REST API + WebSocket |
| **部署** | Docker / Docker Compose + NVIDIA Container Toolkit |

---

## 🚀 适用场景

- **个人创作者**：为 YouTube/Bilibili 视频制作高质量双语字幕
- **字幕组**：批量处理剧集，自动化提取→识别→翻译全流程
- **研究者**：语音识别模型评测、翻译质量对比
- **NAS 用户**：通过 Docker 部署在群晖/Unraid/TrueNAS 上 7×24 运行

## ⚡ 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/SiVeci/EchoSRT.git
cd EchoSRT

# 2. Docker Compose 一键启动
docker compose up -d

# 3. 访问 Web UI
open http://localhost:9000
```

详细步骤请参阅 [快速上手](getting-started/quick-start.mdx)。
