## v1.3.0 (2026-05-19)

- **核心功能**
    - **本地 LLM 翻译引擎**：集成 `llama-cpp-python`，支持通过本地 GGUF 模型进行完全离线的字幕翻译，提供更佳的隐私与零成本方案。
    - **GPU 显存互斥调度**：引入显存锁机制，确保本地语音识别（faster-whisper）与本地大模型翻译任务互斥执行，避免并发加载导致显存溢出；翻译任务在识别任务运行时自动让渡资源，保障系统稳定性。
    - **批量资产清理**：任务工作区新增"批量清理"功能，支持一键删除多个任务的所有关联资产（视频、音频、字幕）。
- **界面与交互**
    - **UI 全面重构**：优化了设置面板与任务流布局，更新了全量系统图标，提升视觉一致性。
    - **标签文本优化**：简化了媒体库映射视频的显示标签，界面更加清爽。
- **架构改进**
    - **翻译引擎解耦**：重构了翻译逻辑，实现 API 模式与本地推理模式的平滑切换。
    - **字幕解析宽容模式**：优化翻译模块的 SRT 解析逻辑，支持宽容的正则匹配方案，增强对非标准或格式略有偏差的字幕文件的兼容性，减少解析失败概率。
- **问题修复**
    - **Hugging Face 令牌参数传递修复**：修复了之前填写的 Hugging Face 令牌参数无法正确传递到模型下载任务中的问题，确保受限模型能够正常下载。
- **文档完善**
    - **CUDA 环境安装指引**：README 新增本地 Python 运行环境下 CUDA 加速依赖的安装说明，降低用户配置门槛。

## v1.2.2 (2026-05-14)

- **性能优化**
    - **内存自动回收**：引入 `TERMINAL_STATES` 机制，任务完成后自动从内存卸载，大幅降低长程运行下的 RAM 占用。
    - **按需状态加载**：后端启动时仅加载活跃任务，历史任务状态转为“磁盘回退读取”模式。
- **鲁棒性增强**
    - **全链路取消适配**：重构 API 识别与 LLM 翻译的异步中断逻辑，确保底层 OpenAI/HTTP 请求在取消时立即切断。
    - **并发冲突修复**：修复了批量取消任务时可能触发的字典迭代大小变化错误。
- **接口与 UI**
    - **状态字段透传**：`GET /api/tasks` 等列表接口新增 `current_step` 字段，支持前端展示历史任务的最终状态。
    - **前端自愈刷新**：增强 `TabWorkspace` 的状态捕获逻辑，支持在任务结束瞬间精准刷新资产图标。

## v1.2.1 (2026-05-13)

- **核心功能**
    - **断点续传与状态持久化**：任务进度实时落盘 (`state.json`)，支持服务端重启后自动恢复，并允许在提取、识别、翻译中断后无缝重试。
- **性能与优化**
    - **全异步调度**：后端资源清理与状态管理重构为异步逻辑，提升高并发效能。
    - **UI 智能重试**：前端界面与轮询机制智能适配异常中断状态，支持快捷断点重试并过滤陈旧数据下发。

## v1.2.0 (2026-05-12)

- **核心功能**
    - **全链路任务中断**：新增全局与单任务中断机制，支持一键强制停止底层处理进程，并自动清理残余临时文件释放资源。

- **架构与稳定性**
    - **跨版本兼容**：重构并发与网络栈，现全面支持 Python 3.9 至 3.13。
    - **高并发修复**：强制采用 `spawn` 进程模式与强引用锁，彻底消除激进 GC 与容器环境引发的任务死锁及僵尸进程。
    - **资源泄漏自愈**：优化任务删除逻辑，同步回收全局事件、Socket 连接及线程锁，解决长程运行导致的服务端资源泄漏。
    - **现代化迁移**：全量迁移至 `httpx` 提升网络性能，使用原生集合类型消除高版本 Python 语法警告。

- **交互优化**
    - **长文件名感知**：优化任务栏布局，新增长文本溢出截断与悬浮提示，保障超长文件名下的界面整洁。

## v1.1.1 (2026-05-11)

- **架构级重构**
    - **多进程隔离**：将 Whisper 推理迁移至独立的 **Worker 子进程**，彻底解决模型卸载引发的 Python 闪退问题。
    - **极致资源回收**：子进程空闲 5 分钟后自动销毁，由操作系统内核强制释放显存。

- **翻译引擎增强**
    - **LLM模型适配**：支持用户自定义 **Max Tokens** 和 **Temperature**，解决长文本思考导致的截断问题。

- **UI交互修复**
    - **实时反馈自愈**：修复原声/翻译图标点亮延迟；引入 1.5s 安全延迟解决 NTFS 文件系统同步竞态。
    - **资产查询加速**：禁用前端资产轮询的浏览器缓存。

- **系统健壮性**
    - **删除逻辑优化**：删除模型前强杀 Worker 进程释放文件锁。
    - **安全防护补齐**：在所有入口添加 `freeze_support()` 以适配 Windows 多进程环境。
