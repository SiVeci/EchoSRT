📝 更新日志

本页记录 EchoSRT 各版本的显著变更。

未发布 (dev)​

🚀 新功能​

  • VAD 语音活动检测：集成 silero-vad，在语音识别前自动裁切静音段，减少无效推理
  • 多引擎支持：语音识别支持 faster-whisper 和在线 API 双引擎切换
  • 批量翻译：LLM 翻译新增 max_concurrent 并发控制，大幅提升长文本翻译速度
  • SRT 排版熔断：翻译结果自动保留原始 SRT 索引和时序，避免格式错乱

🔧 改进​

  • 代理设置从仅在任务启动时生效 → 配置变更时即时刷新环境变量
  • Docker 镜像内置 FFmpeg，不再要求宿主机安装
  • GPU 镜像基于 CUDA 12.2 + cuDNN 8 运行时，降低镜像体积
  • 前端全局日志面板支持自动滚动和级别过滤

🐛 修复​

  • 修复多个 WebSocket 监听器重复订阅导致状态不同步
  • 修复 config.json 部分字段缺失时启动崩溃的兜底逻辑
  • 修复 GPU 版本模型路径查找失败问题
  • 修复在线 ASR 长音频分块后时间戳偏移

v1.0.0 (2026-01)​

🎉 首个正式版本​

  • 架构：FastAPI + Vanilla JS 纯静态前端
  • 音频提取：基于 FFmpeg，支持自定义采样率、编码器
  • 语音识别：
    • 本地引擎：faster-whisper（支持 tiny ~ large-v3 全部蒸馏模型）
    • 云端引擎：OpenAI Whisper API 兼容
  • 字幕翻译：GPT-4o / DeepSeek / 自定义 OpenAI 兼容 API
  • 工作区管理：任务隔离目录，产物持久化
  • Docker 部署：CPU + GPU 双版本镜像
  • 实时进度推送：WebSocket 双向通信
  • 全局代理：内置 HTTP/SOCKS5 代理，解决国内网络问题
  • 配置管理：config.json 运行时读写，前端 GUI 配置面板
